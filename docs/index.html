Privacy Policy for Unicode Text Converter

Effective date: March 29, 2026

Unicode Text Converter is a Chrome extension that converts user-selected text into Unicode-styled text directly in the browser. This Privacy Policy explains what data the extension processes, how it is used, and what the extension does not do.

1. What the extension does

The extension lets users select text on a webpage, preview Unicode style variations, and then copy, insert, or replace that text in supported editable fields. It also provides a popup for manually entering text and converting it locally in the browser.

2. Data the extension processes

The extension may process the following data only when needed to provide its user-facing features:

  • Text selected by the user on a webpage
  • Text entered by the user into the extension popup or inline picker
  • User preference settings, such as the last used style and behavior options

3. How the data is used

The extension uses data only to provide its single purpose: converting text into Unicode-style formats for the user.

  • Selected or entered text is processed locally to generate Unicode-styled output
  • Converted text may be copied to the clipboard when the user explicitly chooses the copy action
  • Preference settings are stored so the extension can remember the user’s choices across sessions

4. Data storage

Preference settings are stored using Chrome extension storage, including chrome.storage.sync when available. This is used only to remember user settings such as:

  • Last used style
  • Default style
  • Preferred action
  • Enable or disable state
  • Other extension behavior preferences

The extension is not designed to store a history of converted webpage text for analytics, profiling, or resale.

5. Clipboard access

The extension writes to the clipboard only when the user explicitly chooses to copy converted text. It does not read clipboard contents for tracking or unrelated purposes.

6. Data sharing and sale

Unicode Text Converter does not sell user data. It does not transfer user-selected text, manually entered text, or preference data to third parties for advertising, analytics, profiling, or other unrelated purposes.

7. Remote services and remote code

The extension is intended to operate locally in the browser. It does not rely on external text-conversion APIs to perform its core conversion features. Executable code is packaged with the extension and is not downloaded remotely for execution.

8. Permissions usage

The extension uses Chrome permissions only for its described user-facing functionality:

  • Website access: to read user-selected text and apply converted text in supported editable fields
  • Storage: to save user preferences
  • Clipboard write: to copy converted text when the user requests it
  • Context menus: to provide right-click actions for selected text

9. What the extension does not do

  • It does not sell user data
  • It does not use user data for advertising or profiling
  • It does not collect financial, health, authentication, or location information
  • It does not use user data to determine creditworthiness or for lending purposes
  • It does not intentionally collect browsing history for analytics or tracking

10. Changes to this policy

This Privacy Policy may be updated if the extension’s functionality changes. Any future updates should remain consistent with the Chrome Web Store disclosure requirements and applicable laws.

11. Contact

For questions about this Privacy Policy, users may contact the publisher through the contact details provided in the Chrome Web Store listing for Unicode Text Converter.
